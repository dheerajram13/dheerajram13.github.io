Devathon (Crypsis Technologies Pvt. Ltd.)
Company Website

This was a remote internship to design and implement a chatbot from scratch. I have developed a Eatable (Restaurant Chatbot) which can:

  • Ask for restaurant recommendation.

  • Locate nearest branch of a particular restaurant chain.

  • Query for timings of a restaurant.

  • Search for restaurants by name, cuisine, location(nearby), budget.

  • Add/remove from list of favorites.

This chatbot was implemented by using the NLP engine: Dialog flow, Zomato API , Python, Flask for backend and integrated to Facebook Messenger.
